## Notice 및 사용법

Notice

  

  1. **Python을 기반으로 제작되었습니다.**

  2. **Linux Mint 22.3 (Ubuntu 24.04), Python 3.12.3 환경에서 제작, 테스트하였습니다.**

  3. **[Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) 과 같은 Interpreter 변환 구조입니다.**

  4. **Windows 환경에서는 Unicode, UTF-8 인코딩 Issue로 한글이 깨지는 문제가 있습니다.**

  5. **Source Code 확장자는 ```.yr```를 사용하며, 일반적인 텍스트 편집기로도 작성&수정할 수 있습니다.**
  
  6. **Google Gemini의 도움을 받아 제작되었습니다.**
     
  > **이 프로그래밍 언어는 실사용이 불가능한 '난해한 프로그래밍 언어'입니다.**
 
  > **튜링 완전성에는 만족하지만 어떤 가치도 없는 쓰레기입니다. 뭔가 해보려는 시도를 권장하지 않습니다.**

  > **굳이 이 쓰레기의 소스코드는 공개하지 않아도 되지 않을까 합니다.**
---
사용법
  1. 본 Repo를 PC에 Save 또는 Clone합니다.

  2. 첨부된 **.whl** 패키지를 ```pip install yareulang-0.1.0-py3-none-any.whl``` 명령어로 설치합니다.

     ( 부디 가상환경에 설치하시길 바랍니다 )
  
  3. ```yareu example.yr```, ```yareu gugudan.yr``` 명령어로 예제 코드를 실행해볼 수 있습니다.
  4. 첨부된 **generator.py**로 입력한 Text를 코드로 자동 변환 & 제작해보실 수 있습니다.
     
     ( ```python generator.py``` 명령어로 실행 )

## Interpreter 구조


| Keyword | Brainfuck | Details |
| :---: | :---: | :--- |
| 야 | > | 포인터를 오른쪽으로 1칸 이동합니다. |
| 르 | < | 포인터를 왼쪽으로 1칸 이동합니다. |
| 야르 | + | 현재 포인터가 가리키는 메모리 칸의 값을 1 증가시킵니다. |
| 야르릉 | - | 현재 포인터가 가리키는 메모리 칸의 값을 1 감소시킵니다. |
| 야르렁 | . | 현재 메모리 칸의 값을 ASCII 코드 문자로 변환하여 출력합니다. |
| 야르킁킁 | , | 사용자로부터 문자 1개를 입력받아 ASCII 코드 값으로 현재 칸에 저장합니다. |
| 야르쫑긋 | [ | 현재 칸의 값이 0이면, 짝을 이루는 야르꼴깍 다음으로 건너뜁니다. |
| 야르꼴깍 | ] | 현재 칸의 값이 0이 아니면, 짝을 이루는 야르쫑긋으로 돌아갑니다. |
| 야르덥석 | | 현재 메모리 칸의 값을 별도의 임시 클립보드 버퍼에 복사합니다. |
| 야르폴짝 | | 클립보드 버퍼에 저장된 값을 현재 메모리 칸에 덮어씁니다. |
| 할렐야루 | | 프로그램의 실행을 즉시 강제 종료합니다. |
